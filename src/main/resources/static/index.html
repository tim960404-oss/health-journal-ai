決策樹邏輯

系統用以下多層決策樹判斷健康風險等級

ROOT: 睡眠時數 (sleep_hours) ├── < 6 小時 │ ├── 步數 (steps) │ │ ├── < 3000 │ │ │ └── 🔴 HIGH（高風險） │ │ └── ≥ 3000 │ │ ├── 心情分數 (mood_score) │ │ │ ├── ≤ 4 → 🔴 HIGH（高風險） │ │ │ └── > 4 → 🟡 MEDIUM（中風險） └── ≥ 6 小時 ├── 步數 (steps) │ ├── < 5000 → 🟡 MEDIUM（中風險） │ └── ≥ 5000 │ ├── 心情分數 (mood_score) │ │ ├── ≥ 6 → 🟢 LOW（低風險） │ │ └── < 6 → 🟡 MEDIUM（中風險）

此決策樹先判斷睡眠時數，再依步數，最後依心情分數，形成三層判斷， 自動輸出低 / 中 / 高風險等級並存入資料庫。
